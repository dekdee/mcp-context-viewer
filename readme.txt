A simple admin tool to validate and view MCP (Model Context Protocol) JSON metadata.

== Description ==

This plugin lets you validate and pretty-print MCP (Model Context Protocol) JSON context in the WordPress admin.

Learn more about MCP at [mcp-hunt.com](https://www.mcp-hunt.com/)

== Installation ==

1. Upload `mcp-context-viewer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit **MCP Viewer** in admin
